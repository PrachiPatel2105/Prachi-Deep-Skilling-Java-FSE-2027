CONTROL STRUCTURES LEARNED:

1. IF / ELSE:
   - IF condition THEN ... END IF;
   - ELSIF for multiple conditions
   - ELSE for default case
   - Useful for checking age, balance, etc.

2. FOR LOOP:
   - FOR variable IN start..end LOOP ... END LOOP;
   - FOR record IN (cursor) LOOP ... END LOOP;
   - Easy to iterate through records
   - Most common in PL/SQL

3. WHILE LOOP:
   - WHILE condition LOOP ... END LOOP;
   - Repeats while condition is true
   - Less common than FOR loop

4. LOOP / EXIT:
   - LOOP ... EXIT WHEN ... END LOOP;
   - Infinite loop with exit condition
   - Used when exit condition complex

5. CURSOR:
   - Selects rows from table
   - Loop through each row
   - Access columns from each row

SCENARIO 1 - AGE DISCOUNT:
Purpose: Apply 1% discount to customers > 60
Logic:
- FOR each customer:
  - IF age > 60:
    - FOR each loan of customer:
      - UPDATE interest_rate - 1%

Key Concepts:
- Nested loops (customer → loans)
- IF statement
- UPDATE in loop
- COMMIT after all updates

Expected Results:
- 5 customers > 60 identified
- 8 loans updated
- Interest rates reduced by 1%

SCENARIO 2 - VIP PROMOTION:
Purpose: Set IsVIP = 'Y' for balance > $10,000
Logic:
- FOR each customer:
  - IF balance > 10000:
    - IF already VIP: print "Already VIP"
    - ELSE: UPDATE IsVIP = 'Y'

Key Concepts:
- Simple loop
- IF/ELSE for eligibility
- Nested IF for existing status
- UPDATE statement
- Multiple conditions

Expected Results:
- 5 customers promoted to VIP
- Updated IsVIP flag
- Status logged

SCENARIO 3 - LOAN REMINDERS:
Purpose: Find loans due within 30 days
Logic:

- FOR each loan:

  - Show urgency level

Key Concepts:

Expected Results:

    -- code
ELSE
    -- code
END IF;

UPDATE:
UPDATE Customers 
SET IsVIP = 'Y'
WHERE CustomerId = customer_rec.CustomerId;

OUTPUT:
DBMS_OUTPUT.PUT_LINE('Message: ' || variable);

COMMIT:

EXCEPTION:
EXCEPTION
    WHEN OTHERS THEN
        ROLLBACK;
        DBMS_OUTPUT.PUT_LINE('Error: ' || SQLERRM);
