JUNIT ASSERTIONS TESTED:

1. assertEquals()
   - Used for comparing values
   - Works with int, String, double, etc
   - Delta parameter for floating point precision
   - Example: assertEquals(8, 5+3)

2. assertTrue() / assertFalse()
   - Check boolean conditions
   - assertTrue() passes if condition is true
   - assertFalse() passes if condition is false
   - Example: assertTrue(5 > 3)

3. assertNull() / assertNotNull()
   - Check if object is null or not
   - assertNull() for null references
   - assertNotNull() for non-null objects
   - Example: assertNotNull(user)

4. assertSame() / assertNotSame()
   - Check object identity (reference, not value)
   - assertSame() - same instance
   - assertNotSame() - different instances
   - Example: assertSame(obj1, obj2)

5. assertArrayEquals()
   - Compare array contents
   - Works with int[], String[], double[], etc
   - Delta parameter for double arrays
   - Example: assertArrayEquals(expected, actual)

6. fail()
   - Force test to fail
   - Used to mark unimplemented code
   - Example: fail("Not yet implemented")

BEST PRACTICES LEARNED:

1. Use Specific Assertions
   GOOD: assertTrue("Value > 0", value > 0)
   BAD: assertEquals(true, value > 0)

2. Include Descriptive Messages
   GOOD: assertEquals("User age", 25, age)
   BAD: assertEquals(25, age)

3. Test Both Cases
   Positive: assertTrue("Valid", isValid(good))
   Negative: assertFalse("Invalid", isValid(bad))

4. Use Delta for Floats
   assertEquals("PI", 3.14, pi, 0.0001)

5. Keep It Clear
   assertNotNull("User exists", user)

ASSERTION SELECTION GUIDE:

Use assertEquals() when:
- Comparing values (numbers, strings)
- Want specific error message
- Testing equality

Use assertTrue() when:
- Testing boolean conditions
- Checking relationships (>, <, ==)
- More readable than assertEquals(true, condition)

Use assertNull() when:
- Checking for null references
- Validating optional values
- Better than assertEquals(null, obj)

Use assertSame() when:
- Testing object identity
- Checking if same instance
- Not just value equality

Use assertArrayEquals() when:
- Comparing arrays
- Need to verify all elements
- More readable than assertEquals(arr1, arr2)
